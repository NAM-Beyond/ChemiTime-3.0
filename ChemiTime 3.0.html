How to use this App
- Click on the buttons to add a new item (top left are INNs, top right are doses, bottom right are dates)
- Right now you have to double click (create the input field then focus that field) to enter new data since I can't manage to create autofocus on the newly created input field yet
- Press "Enter" key in order to validate the data entered in the input field
- It is advised to begin with entering 2 dates before adding INNs, even if that "INNs first" case has been carried
- Doses buttons won't appear until 2 dates are created
- You can use HTML tags, they should be taken into account (works for styling tags and style attributes)
